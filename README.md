# SnowLeopard Vision

SnowLeopard Vision 是一款本地 AI 图片和视频增强工具，支持图片超分、视频超分、视频补帧、超分并补帧。

当前版本：V1.1

## V1.1 更新

- x4plus / x4plus-anime 为 4x 原生模型，应用内已限制为仅可选择 4x，避免 2x / 3x 出现块状错位。
- realesr-animevideov3 更新为推荐使用模型，兼容性好，支持 2x / 3x / 4x 原生倍率，速度较快。
- 修复处理日志在部分 Windows 环境中出现乱码的问题。
