# 雪豹视觉

SnowLeopard Vision 是一款本地 AI 图片与视频增强工具，支持图片超分、视频超分、视频补帧、超分并补帧。

当前版本：V1.3

- [访问官方网站](https://snowleopard-elysia.github.io/SnowLeopard-Vision/)
- [下载当前版本 V1.3](https://github.com/SnowLeopard-Elysia/SnowLeopard-Vision/releases/tag/V1.3)
- [查看全部版本与历史下载](https://github.com/SnowLeopard-Elysia/SnowLeopard-Vision/releases)

## V1.3 更新

- AI 处理显卡与 NVENC 编码显卡分别检测，自动选择可用的 NVIDIA 独显。
- 进度页面显示总进度、当前阶段进度与详细日志。
- 环境准备增加下载校验、压缩包路径保护与中断恢复。
- 改善 Windows 深色模式下的可读性、滚动条、控件状态和小窗口表现。
- 优化界面视觉层次，同时保留原有操作布局。
