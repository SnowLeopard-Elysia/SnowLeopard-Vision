# SnowLeopard Vision

SnowLeopard Vision（雪豹视觉）是一款为零基础用户设计的本地 AI 图片与视频增强工具，支持图片超分、视频超分、视频补帧、超分并补帧。

当前版本：**V1.3**

- [访问官方网站](https://snowleopard-elysia.github.io/SnowLeopard-Vision/)
- [下载 SnowLeopard Vision V1.3](https://github.com/SnowLeopard-Elysia/SnowLeopard-Vision/releases/tag/V1.3)

## 核心功能

- **图片超分**：提高图片分辨率与清晰度，适用于插画、照片和低分辨率图片。
- **视频超分**：逐帧增强视频画面，并输出更高分辨率的视频。
- **视频补帧**：生成真实中间帧，提高视频帧率与运动流畅度。
- **超分并补帧**：在同一任务中同时提升视频清晰度与流畅度。
- **一键准备环境**：自动配置 FFmpeg、Real-ESRGAN 和 RIFE，降低使用门槛。
