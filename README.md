# SnowLeopard Vision

## V1.3 网站验证

- 从本目录运行 `python -m http.server 8765` 后访问 `http://127.0.0.1:8765/`。
- 所有图片与图标必须使用 `assets/` 相对路径。
- 下载按钮指向 GitHub Releases 中的 V1.3 安装版与免安装压缩包。

SnowLeopard Vision 是一款本地 AI 图片和视频增强工具，支持图片超分、视频超分、视频补帧、超分并补帧。

当前版本：V1.3

## V1.3 更新

- AI 处理显卡与 NVENC 编码显卡分别检测并自动选择 NVIDIA 独显。
- 进度页聚焦总进度、当前阶段进度与详细日志。
- 环境准备增加下载校验、压缩包路径保护与中断恢复。
- 改善深色模式可读性、滚动条、控件状态和小窗口表现。
